# 🧪 ADC Theory

🎨 **Analog-to-Digital Conversion**

An ADC samples an analog voltage and produces a digital value. This is the bridge between the physical world and digital logic!

## 🏗️ Types of ADCs
- **Successive Approximation (SAR)**
- **Sigma-Delta**
- **Flash**
- **Dual Slope**

## 📏 Key Parameters
- Resolution (e.g., 8, 10, 12, 16 bits)
- Sampling rate (SPS/kSPS/MSPS)
- Input impedance
- Reference voltage
- Noise and ENOB (Effective Number of Bits)

## 🧠 How It Works
1. Select input channel
2. Start conversion
3. Wait for completion (polling/interrupt)
4. Read digital result

## 🧰 Tips for Success
- Use a stable reference voltage
- Filter analog signals
- Average multiple samples for noise reduction
- Shield analog traces from digital noise
