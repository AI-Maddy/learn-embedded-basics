# 🧪 ADC Peripheral Overview

🎯 **Analog-to-Digital Converter (ADC)**

An ADC converts analog voltages to digital values, enabling microcontrollers to sense the real world. Used in sensors, audio, and more!

## 🌟 Key Concepts
- Resolution (bits)
- Reference voltage
- Sampling rate
- Input channels
- Noise and accuracy

## 🛠️ Typical Usage
- Configure reference and input channel
- Start conversion
- Read digital value
- Apply filtering or averaging

## 🧩 MCU Support
- **AVR, STM32F1, PIC, MSP430, nRF52, CC26xx, GD32VF103, RP2040, C2000**: All have unique ADC modules—see their `.rst` files for details!
