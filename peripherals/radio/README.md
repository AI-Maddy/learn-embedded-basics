# 📡 Radio Peripheral Overview

🌐 **Wireless Communication (Radio)**

Radio modules let your MCU communicate wirelessly—Bluetooth, Zigbee, Sub-GHz, and more!

## 🌟 Key Concepts
- Modulation (FSK, GFSK, OOK)
- Frequency bands (2.4GHz, Sub-GHz)
- Addressing and pairing
- Power management

## 🛠️ Typical Usage
- Initialize radio module (SPI/UART)
- Set address, channel, power
- Transmit/receive packets

## 🧩 MCU Support
- **AVR, STM32F1, PIC, MSP430**: Use external modules (nRF24L01, CC1101, etc.)
- **nRF52, CC26xx**: Built-in radio hardware
