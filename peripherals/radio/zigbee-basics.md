# 🐝 Zigbee Basics

🌐 **Zigbee** is a low-power, mesh networking protocol for IoT devices.

## 🌟 Key Features
- Mesh networking (self-healing)
- Low power consumption
- Secure communication
- Operates in 2.4GHz and Sub-GHz bands

## 🧩 Applications
- Home automation
- Industrial control
- Smart lighting

## 🛠️ How It Works
- Devices join a mesh network
- Data hops between nodes
- Coordinator manages the network

🔑 **Mnemonic:** "Z**I**G**B**E**E: **Z**ero **I**nterference, **G**reat **B**attery, **E**asy **E**xpansion!"

## ✅ Validation Checklist
- Node joins network reliably.
- Routing works after one node drop/rejoin.
- Message latency is acceptable for application needs.

## 🚨 Common Pitfalls
- Poor channel selection in noisy RF environments.
- Weak network key handling/security configuration.
- Coordinator dependency not planned for failures.
