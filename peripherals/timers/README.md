# ⏲️ Timers Peripheral Overview

⏳ **Timers & Counters**

Timers are the heartbeat of embedded systems—used for delays, PWM, timeouts, and more!

## 🌟 Key Concepts
- Timer/counter modes
- Prescalers
- Capture/compare
- Interrupts

## 🛠️ Typical Usage
- Configure timer mode
- Set period and prescaler
- Enable interrupts or polling
- Use for delays, PWM, event counting

## 🧩 MCU Support
- **AVR, STM32F1, PIC, MSP430**: Each has unique timer features—see their `.rst` files for details!
