# 🔌 GPIO Peripheral Overview

🟢 **General Purpose Input/Output (GPIO)**

GPIO pins are the microcontroller's hands and eyes—used for buttons, LEDs, sensors, and more!

## 🌟 Key Concepts
- Input vs. Output
- Pull-up/pull-down resistors
- Debouncing
- Interrupts

## 🛠️ Typical Usage
- Configure pin direction
- Read input or set output
- Handle interrupts for events

## 🧩 MCU Support
- **AVR, STM32F1, PIC, MSP430, nRF52, CC26xx, GD32VF103, RP2040, C2000**: Each has unique GPIO features—see their subfolders and `.rst` files!
